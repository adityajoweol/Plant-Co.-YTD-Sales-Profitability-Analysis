# Plant-Co.-YTD-Sales-Profitability-Analysis
1. Project Title / Headline
🌿 Plant Co.-YTD Sales & Profitability analysis

An interactive and dynamic Power BI dashboard built to analyze Plant Co.’s business performance across sales, quantity, and gross profit metrics—focusing on year-over-year comparisons, country-level performance, profitability trends, and product insights.

2. Short Description / Purpose

The Plant Co. -YTD Sales & Profitability Dashboard is a visually interactive Power BI report designed to help users monitor and evaluate key business metrics such as YTD Sales, PYTD Sales, Gross Profit %, and product performance across multiple countries and product categories.

This dashboard enables business users, analysts, and decision-makers to identify sales trends, profitability gaps, and regional performance patterns through dynamic filtering and KPI-driven analytics.

3. Tech Stack

The dashboard was built using the following tools and technologies:

📊 Power BI Desktop – Main data visualization platform used for dashboard development and reporting.
📂 Power Query – Used for data cleaning, transformation, and shaping before analysis.
🧠 DAX (Data Analysis Expressions) – Used for creating dynamic measures, KPI calculations, PYTD/YTD comparisons, and conditional logic.
📝 Data Modeling – Relationships established among sales, product, account, and time-related tables for efficient filtering and aggregation.
📈 Interactive Visualizations – Includes KPI cards, waterfall charts, treemaps, scatter plots, and trend charts.
📁 File Format – .pbix for dashboard development and .png for dashboard previews/screenshots.

Data Source

The dataset used in this project was sourced from a YouTube Power BI tutorial/project demonstration and adapted for learning, dashboard development, and data visualization practice purposes.

5. Features / Highlights
• Business Problem

Businesses often struggle to quickly evaluate performance across different KPIs, regions, and product categories using raw transactional data.

Key business questions such as:

Which countries are underperforming in sales?
How does current year performance compare to previous year?
Which products contribute most to gross profit?
Are profitability levels aligned with sales growth?
Which accounts generate high revenue but low profitability?

…are difficult to answer without a centralized analytical dashboard.

• Goal of the Dashboard

To build an interactive business intelligence solution that:

Tracks YTD vs PYTD performance dynamically
Enables users to switch between Sales, Quantity, and Gross Profit metrics
Identifies profitable and underperforming regions/products
Supports strategic business decision-making using visual analytics
Provides a clear overview of profitability and operational performance
• Walkthrough of Key Visuals
KPI Cards (Top Section)

Displays key business indicators including:

YTD Performance
PYTD Performance
YTD vs PYTD Difference
Gross Profit Percentage (GP%)

These KPIs provide an instant overview of overall business health.

Metric Selector (Dynamic Slicer)

Interactive slicer allowing users to dynamically switch dashboard metrics between:

Sales
Quantity
Gross Profit

This functionality is powered using DAX SWITCH() and SELECTEDVALUE() measures.

Country Performance Treemap

Visualizes country-wise contribution to YTD vs PYTD performance.

Helps identify:

Top-performing countries
Declining regions
Market distribution patterns
Waterfall Chart – YTD vs PYTD Analysis

Shows increase and decrease in sales performance across countries/products.

Useful for:

Variance analysis
Performance contribution tracking
Identifying major drivers of business growth or decline
Monthly Sales Trend Chart

Compares YTD and PYTD monthly performance trends.

Provides insights into:

Seasonal trends
Monthly growth patterns
Revenue fluctuations over time
Account Profitability Segmentation (Scatter Plot)

Analyzes relationships between:

Gross Profit %
Sales Value

Helps segment accounts into:

High-value profitable customers
Low-margin accounts
Growth opportunity segments
